# Project

**Sunrayz** is an ongoing development game engine oriented for creation of visual novels and similar games.

## Name

Sunrayz is a name of triple wordplay: "sun-raise", "sun rays", and "Raylib". The final -z intentionally marks the phonetic ambiguousness.

## The End Goal

To provide players and visual novel developers with stable, optimised, and lightweight engine to read and make visual novels with.

## Programming Languages

At the current moment (01.05.2026), the are three programming languages used in the project:
- C17: the engine itself, it's core heart
- Lua: apps and games written with the engine
- Python: the build system (considered to replace with Lua eventually)

Another programming language is planned, Sunrayz's own game-oriented and user-friendly.
